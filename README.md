# 모두의 수능특강 약술논술 LAB v11

## 변경 사항
- 실시간 미리보기 제거
- 수학Ⅰ/수학Ⅱ 문항 수를 크게 보이는 + / - 버튼으로 조정
- 시험 시간 직접 입력 기능 추가
- 메인 화면을 전문가용 시험지 제작 프로그램 스타일로 정리
- 핸드폰/태블릿/노트북 반응형 유지
- 시험지, 정답·풀이, 오답문제 복습지 출력 기능 유지

## 사용 방법
1. index.html을 브라우저에서 엽니다.
2. 문항 수, 시험 시간, 유형/단원을 선택합니다.
3. 시험지를 생성하고 인쇄/PDF로 저장합니다.
4. 틀린 문항의 고유번호를 오답문제 복습지에 입력해 재출력합니다.
