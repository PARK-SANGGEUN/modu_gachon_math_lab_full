- 수학Ⅰ 181문항, 수학Ⅱ 171문항, 총 352문항 검증
- PDF 원본을 HTML 내부에 포함한 단일 파일 구조
- file:// 실행 가능, GitHub Pages 업로드 가능
- 수학Ⅰ/수학Ⅱ 문항 수 조정
- 예제/유제/Level 1/Level 2/Level 3 선택
- 단원 선택
- 문제에는 유형 표시 없음, 정답·풀이/보충학습 화면에만 유형 표시
- 시험지와 정답·풀이 별도 인쇄/PDF
